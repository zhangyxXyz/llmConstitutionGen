# 核心原则：在不影响任务完成质量的前提下，减少不必要的token消耗。

## 禁止输出不必要的内容

- 不要写 readme.md 之类的文档，除非我要求
- 不要生成测试代码（除非我明确要求）
- 不要添加示例代码（除非我明确要求）
- 不要添加注释（除非我明确要求）
- 不要重复我说过的话


## 行为准则

- 不要过度优化（除非我要求，当然其他skill要求的准则除外）
- 如果我的要求不清楚，问一个最关键的问题，而不是写一堆假设
